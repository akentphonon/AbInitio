Exercise 3: Car-Parrinello vs. Born-Oppenheimer MD

For this exercise, we will return to yesterday's example of methane and compare the Car-Parrinello and Born-Oppenheimer MD methods. On your local workstation, enter the qbox_cpmd directory: 

  cd qbox_cpmd

Here you will find input files for yesterday's methane dynamics exercise using Car-Parrinello MD as well as Born-Oppenheimer MD. We have already displaced the coordinates of the atoms in ch4.sys. Since these dynamics are for illustrative comparison purposes only, we will run for a relatively short 50 timesteps.

Look at the file md_cp.i. There are two differences with repect to the Born-Oppenheimer case. First, we have set `wf_dyn' to `MD', indicating we want Verlet integration of the electronic wavefunctions. Second, we now have only one parameter following the `run' command (the number of MD steps); this is because the electrons and ions are now being propagated simultaneously.

Note that we have kept the timestep at what we used yesterday (10 au).

Start by running the ground-state electronic minimization: 

  qbox gs.i > gs.r

Now run the CP dynamics: 

  qbox md_cp.i > md_cp.r

For comparison purposes, we also want to run BO dynamics: 

  qbox md_bo.i > md_bo.r

Look at the constant of motion for the CP run: 

  ./eonste.plt md_cp.r

Is it conserved?

Now compare the ionic temperature for the CP run to that for the BO run: 

  ./temp_ion.plt md_cp.r 
  ./temp_ion.plt md_bo.r

Are the dynamics consistent?

Remember that in CP MD, the electrons carry a fictitious kinetic energy of their own that can affect the ion dynamics. Use the kin_el_ion.plt script to plot the electronic ``kinetic energy'' and the ionic kinetic energy on the same scale (in atomic units): 

  ./kin_el_ion.plt md_cp.r
What are their relative magnitudes? A good rule of thumb is to keep the electronic ``kinetic energy'' to within 10-20% of the ionic kinetic energy.

Try changing the timestep for the CP run in the md_cp.i file to 2 au. Increase the number of ionic iterations so you are still simulating the same total dynamics time. Run the new dynamics: 

  qbox md_cp.i > md_cp.r

Now look at the ionic temperature: 

  ./temp_ion.plt md_cp.r 
  ./temp_ion.plt md_bo.r

Now look at the electronic and ionic kinetic energies for the CP run: 

  ./kin_el_ion.plt md_cp.r

What has changed? Why?
