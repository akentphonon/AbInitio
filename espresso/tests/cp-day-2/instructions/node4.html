  • The slope of the mean squared displacement can be used to extract the diffusion coefficient. To do so, calculate the slope of the MSD for the liquid regime only at 3000 K. Load the file in xmgrace and select the relevant regime by choosing Edit -> Regions -> Define. Change `Region type' to `In Horiz Range' and then select the range of interest on the graph.

    Then calculate the slope in that regime by choosing Data -> Transformations -> Regression. Under `Restrictions' select `Region 0' to include only the range of interest. The output in the popup console will give the slope, which will be in units of Bohr/iteration.   • The Lindemann criterion is a rule of thumb for determining melting points based on the RMS amplitude of vibration in the solid (see Jin et al., PRL 87, 055703). It predicts that the system will melt when this quantity reaches about 10% of the nearest-neighbor distance for the perfect crystal.
